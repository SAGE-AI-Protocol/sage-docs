# Sage.ai 투자자 원페이저

> **"모든 개인 투자자에게 월가 수준의 AI 멘토를"**

---

## 문제

암호화폐 투자자 **5.6억 명**이 직면한 3가지 문제:

| 문제 | 현실 |
|------|------|
| **정보 과부하** | 24시간 시장, 끊임없는 뉴스 - 무엇을 믿어야 할지 모름 |
| **신뢰의 부재** | 리딩방은 70%이상이 사기, 연간 피해액 1,266억원+ |
| **모니터링 불가** | 자는 동안 급락, 급등 → 패닉셀, FOMO |

---

## 솔루션: Sage.ai (가칭)

**암호화폐 투자자를 위한 24시간 AI 챗봇 멘토**

ChatGPT처럼 대화하지만, 실시간 시세를 알고, 시장 급변 시 먼저 알려주며, AI의 조언 성과를 투명하게 추적할 수 있습니다.

### 사용 시나리오

```
🌙 새벽 3시, 비트코인 10% 급락

    ↓ AI가 먼저 Push 알림 전송
    "비트코인이 10% 급락했습니다. 분석을 확인해보세요."

    ↓ 앱을 열면 AI가 상황 분석 제공
    "공포 지수 25, 과매도 구간입니다.
     장기 관점에서 분할 매수를 고려해볼 만합니다."

    ↓ 대화로 추가 질문
    "지금 사도 될까?" → AI가 리스크와 기회 분석

    ↓ AI 조언을 가상 포트폴리오에 저장
    "담아두기" 클릭 → 나중에 AI 조언의 실제 성과 확인 가능
```

### 핵심 가치

| 가치 | 설명 |
|------|------|
| **24시간 모니터링** | 자는 동안에도 AI가 시장을 보고, 급변 시 먼저 알림 |
| **신뢰할 수 있는 정보** | 실시간 시세 기반 답변, 거짓 정보(환각) 1% 미만 |
| **검증 가능한 성과** | AI 조언을 가상으로 추적 → "이 AI 말 믿어도 되나?" 직접 확인 |

## 핵심 차별화: 성과 추적 포트폴리오

> 데이터를 보여주는 서비스는 많지만, **AI 조언의 성과가 투명하게 기록되는** 서비스는 없습니다.

```
[Phase 1] 가상 포트폴리오
AI 조언 → "담아두기" → 성과 추적 → "이 AI 믿을 만하네"

[Phase 2] 실제 거래 연결
거래소 API 연동 → AI 조언 → 원클릭 실제 매매
```

### 왜 기존 서비스로는 안 되나?

| 기존 대안 | 문제점 | Sage.ai |
|-----------|--------|---------|
| **ChatGPT** | 실시간 시세 모름, **먼저 알려주지 않음** | ✅ 시장 급변 시 AI가 먼저 알림 |
| **리딩방** | 70%가 사기, **성과 검증 불가** | ✅ 모든 조언의 성과가 기록됨 |
| **트레이딩뷰** | 데이터만 제공, **해석은 내가** | ✅ AI가 해석 + 조언까지 |
| **로보어드바이저** | 암호화폐 미지원, 대화 불가 | ✅ 암호화폐 특화 + 자연어 |
